WabbitWeb

This is the WabbitWeb payload.

Features:

  • Payload Editor: Create instantly runnable payloads in-browser!
  • Payload Launcher: Launch payloads you created with the Payload Editor!
  • SMB Launcher: Launch an SMB server to WabbitWeb's folder to edit it during runtime!
  • Command Line: Pass commands straight to the Bunny and receive output - all in-browser!
  • Shutdown: Shut down WabbitWeb cleanly with a press of a button!
